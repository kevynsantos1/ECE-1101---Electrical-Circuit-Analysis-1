For the first part of this lab, we used superposition to be able to get our values. This means
we turned one and left one on, solved for our values, then turned off the other one and left the other
one on, solved for our values, and then added those values together to get a total. For the second part
of this lab, we utilized Thevenin’s theorem which states that any two-port linear network may be
replaced by a single voltage source with an appropriate internal resistance.
