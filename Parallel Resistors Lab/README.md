In this lab we arranged our resistors of different values in parallel instead of series. When
you want to combine resistance of resistors that are parallel you multiply the resistor values on top then
divide by the resistor values added. Our voltage was 8 volts and we used that to get current which is
voltage divided by resistance. We got power by multiplying voltage by measured current and got
deviation by Theory current minus Measured current divided by Theory current multiplied by 100. It is
important to note that we did not have a 3.3kΩ resistor so we used a 3.9kΩ resistor instead.
