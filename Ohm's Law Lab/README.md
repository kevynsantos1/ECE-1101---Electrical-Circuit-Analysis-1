In this lab we are going to be using and verifying Ohm’s Law. Ohm’s Law states that we can
get voltage by multiplying current by resistance. We will be using three different types of resistors 1k,
3.9k, and 8.2k. We will be measuring the resistance of all three types and then using that to get the
power. We used I=V/R which was used to find current. Then we used P=V*I to get the power. Lastly, we
used Dev=(It-Im/It)*100 to get the deviation.
